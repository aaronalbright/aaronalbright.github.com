Current

Project Pembroke

Through an independent study at UF, I'm building an app that helps users understand their car's language a little better. The app uses dashboard warning lights to guide users through diagnosing their cars and inform them using third-party of the consequences of car maintenance neglect.

ONA16 Connect

Think "Seven Degrees of Kevin Bacon" but with Twitter and ONA conference attendees. I co-created this app with Charlie Smart at the 2016 Online News Association Student Newsroom. In about 48 hours, we built the app using the Twitter API, Python, Flask and a lot of D3.

The Alligator mobile redesign

Last year, as the online managing of UF's student newspaper, The Independent Florida Alligator, I redesigned the website's mobile view to be more user friendly. The site was always responsive, but the mobile view wasn't optimized for smaller screens.


Top-100 PS4 Games

This simple Python and Flask project uses data scraped from Metacritic, Amazon and HowLongToBeat to compile a list of the top-rated Playstation 4 games that includes an Amazon link and information on approximately how long it will take to finish.
